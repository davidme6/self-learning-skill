## 核心规则（必须遵守！）

### 🚫 绝对不要创建重复章节！

**错误示范：**
```
03 第三章 继续消费，消费万元_优化版.txt  ← 旧版本
03 第三章 医院消费，消费十万元_优化版.txt ← 新版本
```

**正确做法：**
```
03 第三章 继续消费，消费万元.txt  ← 直接用新版本覆盖
```

## 更新流程（严格执行）

### 1. 修复章节后
```powershell
# 1. 检查是否有重复
Get-ChildItem "novel/神豪：从消费返利开始当世界首富" -Filter "0*.txt" | 
  Group-Object { $_.Name.Substring(0,2) } | 
  Where-Object { $_.Count -gt 1 }

# 2. 如果有重复，删除旧版本，保留最新
$files = Get-ChildItem "novel/神豪：从消费返利开始当世界首富" -Filter "03*.txt" | 
  Sort-Object LastWriteTime -Descending
Remove-Item $files[1].FullName -Force  # 删除旧的

# 3. 验证无重复
Get-ChildItem "novel/神豪：从消费返利开始当世界首富" -Filter "0*.txt" | 
  Group-Object { $_.Name.Substring(0,2) } | 
  Where-Object { $_.Count -gt 1 }  # 应该返回空
```

### 2. 更新原则
- ✅ **直接用新版本覆盖旧版本**
- ✅ **文件名保持简洁**（不要 `_优化版`、`_v2` 等后缀）
- ✅ **每个章节号只对应一个文件**
- ✅ **删除所有备份文件**

### 3. 命名规范
```
❌ 错误：
01 第一章 标题_优化版.txt
01 第一章 标题_v2.txt
01 第一章 标题_修复版.txt

✅ 正确：
01 第一章 标题.txt
```
