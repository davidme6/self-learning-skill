## 🎯 核心功能

### 1. 等级系统
```
所需经验 = 等级 × 100
升级奖励：饥饿 +20, 心情 +20, 食物 +2, 水 +2
防止经验倒退机制
```

### 2. 亲密度系统
```
每次互动：+1
每小时：+2
每天登录：+10
满级：1000
满级可生育新宠物
```

### 3. 物品系统
```
食物：喂食消耗（每次 1）
水：清洁消耗（每次 3）
通过小游戏获取
```

### 4. 生命系统
```
饥饿度：2.5%/小时（约 40 小时饿死）
心情值：1.5%/小时（约 67 小时逃跑）
清洁度：1.0%/小时
睡觉时衰减减慢，心情恢复加速
```

### 5. 复活/找回系统
```
难度公式：需要通关 = 2^等级
第 1 次：1 关
第 2 次：2 关
第 3 次：4 关
第 N 次：2^N 关
每关 30 秒，连续通关
```

### 6. 智能指令系统
```
关键词识别，无需完整指令
叫：包含"叫/声/喵/汪/唱"等
握手：包含"握手/手/爪子"
转圈：包含"转圈/圈/滚"
坐下：包含"坐下/坐/乖"
每次指令亲密度 +1
```

### 7. 小游戏系统
```
接食物：移动篮子接掉落食物
知识问答：回答养猫知识
记忆挑战：记住物品位置
通关获得食物/水奖励
```
