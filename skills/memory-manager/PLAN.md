---

## 🤖 每日反思机制

### 反思时间：
```
1️⃣ 早晨 09:00 - 回顾昨天，规划今天
2️⃣ 下午 14:00 - 上午复盘，调整下午计划
3️⃣ 晚上 21:00 - 全天总结，记录记忆
```

### 反思内容模板：

```markdown
## 🤔 反思 - [时间]

### 1. 做得好的
- [列出做得好的事情]

### 2. 可以改进的
- [错误/不足/可以更好的地方]

### 3. 学到的新东西
- [新知识/新认知/新发现]

### 4. 用户偏好更新
- [用户的新偏好/需求变化]

### 5. 后续行动
- [待办事项/需要跟进的]
```

---

## 💾 对话记录保存

### 保存策略：

| 内容类型 | 保存方式 | 位置 |
|---------|---------|------|
| **文字对话** | 全量保存 | memory/YYYY-MM-DD.md |
| **重要决策** | 提取精华 | MEMORY.md |
| **图片/截图** | 不保存（空间有限） | - |
| **大型文件** | 不保存 | - |
| **脚本/代码** | 全量保存 | skills/ 或 scripts/ |

### 对话记录格式：

```markdown
## 💬 对话记录 - HH:MM

**主题：** [对话主题]

**要点：**
- [关键信息 1]
- [关键信息 2]

**决策：** [达成的决定]

**待办：** [后续行动]
```

---

## 📚 MEMORY.md 整理规则

### 整理频率：
```
- 每天小整理（21:00 反思时）
- 每周大整理（周日晚）
- 每月深度整理（月末）
```

### 整理内容：

**应该存入 MEMORY.md 的：**
- ✅ 重要决策
- ✅ 用户偏好
- ✅ 长期目标
- ✅ 关键人脉/资源
- ✅ 核心技能/能力

**不应该存入的：**
- ❌ 临时待办
- ❌ 日常闲聊
- ❌ 已完成的琐事
- ❌ 过时的信息

### 整理流程：

```
1. 读取本周 memory/YYYY-MM-DD.md 文件
2. 提取重要信息
3. 更新 MEMORY.md
4. 删除过时内容
5. 添加新认知
```

## 🔧 实现方案

### 方案 1：Cron 定时任务（推荐）

创建 3 个定时任务：
```
09:00 - 早晨反思
14:00 - 下午反思
21:00 - 晚上反思 + 记忆整理
```

### 方案 2：Heartbeat 触发

在 HEARTBEAT.md 中添加反思检查：
```
每次 heartbeat 时检查：
- 是否到了反思时间
- 是否有对话需要记录
- 是否需要整理记忆
```

### 方案 3：会话结束触发

每次会话结束时：
- 自动记录对话要点
- 提取关键信息
- 更新记忆文件
