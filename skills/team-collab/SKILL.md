## 🎯 三种协作模式

### 模式一：角色扮演（快速模式）
- **速度快**：一次对话完成全部工作
- **成本低**：单次 API 调用
- **适合**：中小型项目、快速迭代

### 模式二：模型切换（专业模式）
- **专业分工**：真正调用不同模型
- **质量最高**：每个阶段用最擅长的模型
- **适合**：大型项目、高要求场景

### 模式三：混合模式（推荐⭐）
- **兼顾效率与质量**：开发用模式一，审查用模式二
- **性价比高**：关键环节专业，非关键环节快速
- **适合**：软件开发、编程项目
