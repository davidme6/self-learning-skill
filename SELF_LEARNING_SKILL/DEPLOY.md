# 🚀 Self-Learning Skill 部署指南

## 📦 GitHub 部署

### 方式 1：手动上传（推荐）

**步骤**：

1. **创建新仓库**
   - 访问：https://github.com/new
   - 仓库名：`self-learning-skill`
   - 可见性：Public
   - 描述：自我学习与迭代技能系统

2. **上传代码**
   ```bash
   # 进入项目目录
   cd SELF_LEARNING_SKILL
   
   # 初始化 Git（如未初始化）
   git init
   
   # 添加所有文件
   git add .
   
   # 提交
   git commit -m "feat: 创建自我学习与迭代技能 v1.0"
   
   # 关联远程仓库
   git remote add origin https://github.com/YOUR_USERNAME/self-learning-skill.git
   
   # 推送
   git push -u origin master
   ```

3. **完善仓库**
   - 添加 Topics：`ai`, `learning`, `self-improvement`, `skill-system`
   - 设置 Website（如有）
   - 添加 License

### 方式 2：GitHub Desktop

## 🎯 ClawHub 部署

### 上传到 ClawHub

**步骤**：

1. **访问 ClawHub**
   - 网址：https://clawhub.com
   - 登录账号

2. **创建新技能**
   - 点击"创建技能"
   - 填写信息：
     - 名称：Self-Learning Skill
     - 分类：AI 助手/学习工具
     - 标签：`self-learning`, `ai-skill`, `productivity`
     - 简介：持续学习、识别不足、主动增强、迭代进化

3. **上传代码**
   - 方式 A：GitHub 仓库链接
   - 方式 B：直接上传 ZIP 包

4. **提交审核**
   - 等待 ClawHub 审核
   - 审核通过后发布
