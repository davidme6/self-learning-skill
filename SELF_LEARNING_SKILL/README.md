# 🤖 Self-Learning Skill - 自我学习与迭代技能

> 持续学习、识别不足、主动增强、迭代进化

---

## 🎯 技能简介

这是一个**自我驱动的学习系统**，让 AI 助手能够：

1. **主动识别**能力不足
2. **自动触发**学习流程
3. **持续追踪**成长进度
4. **量化评估**学习效果
5. **接受监督**用户随时控制

---

## 📦 核心功能

### 1. 能力评估矩阵

8 大维度全面评估：
- 技术能力（代码/架构/性能）
- 学习能力（掌握/迁移/解决）
- 项目能力（理解/交付/时间）
- 沟通能力（表达/确认/建议）

### 2. 自我检测机制

- **任务后复盘**：每次任务后 5 问
- **每日总结**：22:00 自动执行
- **每周评估**：能力值对比

### 3. 多渠道学习

- GitHub 开源项目
- MDN 官方文档
- 技术社区（掘金/知乎）
- 优秀作品分析

### 4. 进度追踪

- 学习时长统计
- 代码产出追踪
- 能力成长曲线
- 项目进展监控

### 5. 用户控制权

用户随时可以：
- 叫停学习
- 纠正方向
- 调整目标
- 查看进度

## 🚀 使用方法

### 激活技能

技能已内置，自动执行。

### 查看进度

```bash
# 查看学习记录
cat learning/progress-tracker.md

# 查看能力评估
cat .agents/skills/self-learning/SKILL.md
```

### 用户控制

**指令示例**：
- "暂停学习" → 立即停止
- "继续学习" → 恢复学习
- "换个方向" → 调整主题
- "我来看" → 展示进度
- "学这个" → 指定内容
